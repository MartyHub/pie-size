Pie-Size
========

Simple disk space analyzer made with :


- [Node.js](http://nodejs.org/)
- [Socket.IO](http://socket.io/)
- [Highcharts](http://www.highcharts.com/)

A browser window should be opened once the application started :

`node server.js`
